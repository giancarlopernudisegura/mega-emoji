# mega-emoji

## Installation
```sh
git clone https://github.com/giancarlopernudisegura/mega-emoji.git
pip install -e mega-emoji
```

## Usage
```bash
split-image [OPTIONS] IMAGE
```
### Options
| Parameter              | Description |
|------------------------|-------------|
| `-s`, `--size INTEGER` | Grid size   |
| `-v`, `--verbose`      |             |
